# dvrk_auto_path_finder

1. Use Coppelia scene **IDK WHICH ONE** and select start simulation
2. Run the following code
```
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python image_to_coordinates.py
```
